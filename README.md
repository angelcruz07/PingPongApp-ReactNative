## Como iniciar

1. Clona el proyecto

```bash
git clone https://github.com/angelcruz07/ping-pong-expo
```

2. Instala las depenciencias

   ```bash
   npm install
   ```

3. Inicia la aplicacion

   ```bash
    npx expo start
   ```
